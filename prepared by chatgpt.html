Learning HTML: Tables and Lists

Welcome to this learning resource where we'll explore how to create and style tables and lists using HTML. By the end of this page, you'll understand how to use various HTML attributes and elements to build structured content on a webpage.

TABLE USING CELLPADDING AND CELLSPACING

In HTML, cellpadding adds space inside the table cells, and cellspacing adds space between the cells. This can make your table more readable by preventing the content from being too close together.

Here's an example:

Student ID Student Name
101 Sudhakar
102 Sagar
103 Sudha Murthy

TABLE USING CELLSPACING ONLY

In this example, the cellpadding is disabled, so only the cellspacing is applied:

Student ID Student Name
101 Sudhakar
102 Sagar
103 Sudha Murthy

TABLE USING CELLPADDING ONLY

Here, the cellspacing is disabled, so only the cellpadding is applied:

Student ID Student Name
101 Sudhakar
102 Sagar
103 Sudha Murthy

HTML LISTS

Lists are a great way to organize content. HTML provides two main types of lists:

Ordered Lists

Ordered lists use numbers or letters to order list items:

  1. This is a list item
  2. This is another list item
Different Numbering Styles:
  1. Item a
  2. Item b
  1. Item i
  2. Item ii
  1. Item 5
  2. Item 6

Unordered Lists

Unordered lists use bullets to list items:

  • This is a list item
  • This is another list item
Different Bullet Styles:
  • Item with a circle bullet
  • Item with a square bullet

Note: The attribute type="cube" is not a valid HTML option and won't change the list style.
