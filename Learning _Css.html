Selectors :

Selectors are used to select particular element or particular portion .

Types :

  • Name selector :Selects HTML elements based on their tag names (e.g., p, h1, div).Usage: This selector styles all instances of the specified element type on the page.Syntax: Simply use the element's name.
  • Class selector :Selects elements that have a specific class attribute. Classes can be used on multiple elements, making them suitable for styling groups of elements,Usage: Use for styling multiple elements with the same class. Syntax: Use a . followed by the class name
  • Id selector :Selects an element based on its unique id attribute. IDs are unique within a page and should be used only once per element,Usage: Use when you need to style a single, specific element.Syntax: Use a # followed by the element's ID
  • Universal selector :Selects all elements on the page,Usage: Useful for applying a style to every element on the page, like resetting margin and padding. Syntax: An asterisk "*".

Inclusion in HTML :

  1. Inline :Inline CSS involves writing CSS rules directly inside the HTML elements using the style attribute. It applies styles directly to a specific element.
    • Usage: Ideal for quick fixes or unique styling for a single element but not recommended for large-scale styling due to poor maintainability.
  2. Embedding :Embedded CSS (or internal CSS) refers to styles written within a style tag placed in the head section of the HTML document. It applies to the entire HTML document.
    • Usage: Useful for applying styles to a single HTML page. It keeps styles within the same file but still separates them from the content
  3. Linking :Linking CSS involves using an external .css file and connecting it to the HTML document with a tag inside the . This approach keeps the HTML and CSS code separate, promoting better organization and reusability.
    • Usage: Best for larger projects where the CSS needs to be reused across multiple HTML files. This method allows for a clean and maintainable structure.
  4. Importing :Exporting CSS generally refers to creating a separate CSS file (.css) that is referenced in multiple projects or documents. It's essentially another term for external CSS but emphasizes reusability across different applications or environments.
    • Usage: Used when you want to share CSS rules across various web applications or ensure consistency in styling multiple projects by exporting the CSS file and including it wherever necessary.

Background-image :

 Sets the background image for an element.

Type of attributes we can use in Background-image are :

Types :

  1. background-repeat :
    • Values :
    • 'repeat' (default): Repeats the image both horizontally and vertically.
    • 'repeat-x': Repeats the image horizontally only.
    • 'repeat-y': Repeats the image vertically only.
    • 'no-repeat': The image is not repeated.
  2. 'background-size' :Defines the size of the background image.
    • Values :
    • 'auto': Keeps the original size of the image.
    • 'cover': Scales the image to cover the entire element, cropping as necessary.
    • 'contain': Scales the image to fit within the element, maintaining aspect ratio.
    • Specific sizes like 100px, 50%, etc.
  3. 'background-position': Sets the starting position of the background image within the element.
    • Values :
    • Keywords: top, bottom, left, right, center.
    • Coordinates: 50px 100px, 20% 30%.
  4. background-attachment :Specifies whether the background image scrolls with the page or is fixed.
    • 'scroll': The background image scrolls with the page.
    • 'fixed': The background image is fixed in place.
    • 'local': The background scrolls within the element it is applied to.
  5. 'background-clip':Defines the area within which the background image is visible.
    • 'border-box': The background extends to the outer edge of the border.
    • padding-box: The background extends to the edge of the padding box.
    • content-box: The background is only visible within the content area
  6. 'background-origin' :Sets the positioning area of the background image.
    • Values :
    • 'border-box': Positions the background relative to the border box.
    • 'padding-box': Positions the background relative to the padding box
    • 'content-box': Positions the background relative to the content box.
  7. 'background-color': Sets a background color that appears behind the background image or when the image is not visible.
  8. background-blend-mode :Specifies how the background image blends with the background color or other background images.
    • normal, multiply, screen, overlay, darken, lighten, etc.
