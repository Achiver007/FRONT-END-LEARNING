Phrase Tags

1. EMPHASIS (<em>):

Definition: The <em> tag is used to emphasize text, typically rendering it in italics. It indicates that the enclosed text has stress or importance.

Example:

This text is emphasized.

Output: This text is emphasized.

2. ADDRESS (<address>):

Definition: The <address> tag is used to provide contact information for the author/owner of a document or article. It typically renders in italics and often includes an email address, phone number, or physical address.

Example:

Please reach me at:

3-140/1, Venkateswara Hills Colony
.

Output: Please reach me at: 3-140/1, Venkateswara Hills Colony.

3. MARKED TEXT (<mark>):

Definition: The <mark> tag highlights text by changing its background color to yellow, indicating it’s important or relevant.

Example:

This is my book.

Output: This is my book (with yellow background).

4. ABBREVIATION (<abbr>):

Definition: The <abbr> tag is used to define an abbreviation or acronym. The title attribute is often used with this tag to provide the full form when the user hovers over the abbreviation.

Example:

I love to work at CIL.

Output: I love to work at CIL (hovering over "CIL" shows "COAL INDIA LIMITED").

5. STRONG (<strong>):

Definition: The <strong> tag is used to indicate strong importance, seriousness, or urgency. It typically renders text in bold.

Example:

This is my life.

Output: This is my life.

6. QUOTATION (<q>):

Definition: The <q> tag is used for short inline quotations. Browsers typically enclose the quoted text in quotation marks.

Example:

The teacher said, Knowledge is power.

Output: The teacher said, “Knowledge is power.”

Meta Data and Comments

META DATA: Meta data refers to data about data. It is often used in <meta> tags in the <head> section of an HTML document to provide information like keywords, description, author, etc., which is not visible on the web page but helps with SEO and other purposes.

COMMENTS (<!-- ... -->):

Definition: Comments in HTML are enclosed within <!-- ... -->. They are not displayed on the webpage but help developers understand the code. Comments are often used to explain the purpose of certain parts of the code or to temporarily disable code without deleting it.

Example:

<!-- This is a comment and won't be displayed on the webpage -->

Output: (Nothing is displayed on the page).

Image Tag

IMAGE TAG (<img>):

Definition: The <img> tag is used to embed images in an HTML document. The src attribute specifies the path to the image, alt provides alternative text if the image cannot be displayed, border sets the border thickness, and align controls the alignment of the image relative to surrounding text.

Example:

<img src="Images/logo.jpg" alt="logo" border="2" align="left">

Output: An image from Images/logo.jpg is displayed with a border of 2 pixels, aligned to the left.

HTML Hyperlinks Cheat Sheet

1. What is a Hyperlink?

A hyperlink (or link) is a clickable element in a web page that directs users to another page or resource, such as a document, image, or another website.

2. Basic Hyperlink Syntax

<a href="URL">Link Text</a>

<a href="https://www.example.com">Visit Example</a>

3. Types of Links

3.1. Absolute Links

<a href="https://www.example.com">Example Website</a>

<a href="https://www.example.com">Example Website</a>

3.2. Relative Links

<a href="/about.html">About Us</a>

<a href="/about.html">About Us</a>

3.3. Anchor Links (In-Page Navigation)

<a href="#section1">Go to Section 1</a>

<a href="#section1">Go to Section 1</a>
...
<h2 id="section1">Section 1</h2>

4. Opening Links in a New Tab

<a href="https://www.example.com" target="_blank">Open in New Tab</a>

<a href="https://www.example.com" target="_blank">Open in New Tab</a>

5. Mailto Links (Email Links)

<a href="mailto:someone@example.com">Send Email</a>

<a href="mailto:someone@example.com">Send Email</a>

6. Adding Tooltips to Links

<a href="https://www.example.com" title="Visit Example Site">Example</a>

<a href="https://www.example.com" title="Visit Example Site">Example</a>

7. Linking to Files

<a href="document.pdf" download>Download PDF</a>

<a href="document.pdf" download>Download PDF</a>

8. Styling Links with CSS

/* Unvisited link */
a {
    color: blue;
}

/* Visited link */
a:visited {
    color: purple;
}

/* Mouse over link */
a:hover {
    color: red;
}

/* Selected link */
a:active {
    color: green;
}

9. Best Practices

  • Use Descriptive Text: Make sure the link text is descriptive of the destination.
  • Check Links: Ensure all links work and lead to the correct destination.
  • Use Proper Contrast: Make sure links are easily visible and accessible.


HTML Frames Guide

1. What are HTML Frames?

HTML frames are used to divide a browser window into multiple sections, each of which can load a separate HTML document. Frames were widely used in older websites but are now considered obsolete and replaced by more modern layout techniques such as <div> elements with CSS.

2. Basic Frame Structure

The basic structure of frames is created using the <frameset> and <frame> tags:

<frameset cols="50%,50%">
    <frame src="left.html">
    <frame src="right.html">
</frameset>

In this example, the browser window is divided into two columns.

3. Attributes of <frameset>

3.1. cols

Specifies the width of each column in the frameset. You can use percentages, pixel values, or the wildcard * to allocate remaining space.

<frameset cols="25%,*,25%">
    <frame src="left.html">
    <frame src="main.html">
    <frame src="right.html">
</frameset>

3.2. rows

Specifies the height of each row in the frameset, similar to cols.

<frameset rows="50%,50%">
    <frame src="top.html">
    <frame src="bottom.html">
</frameset>

4. The <frame> Tag

The <frame> tag is used within a <frameset> to define each frame.

<frame src="page.html" name="framename" scrolling="yes" noresize>
  • src: Specifies the URL of the document to display in the frame.
  • name: Assigns a name to the frame, which can be used to target it with links.
  • scrolling: Controls the scroll bars in the frame (yes, no, or auto).
  • noresize: Prevents the user from resizing the frame.

5. Nested Framesets

You can nest framesets to create more complex layouts.

<frameset rows="50%,50%">
    <frame src="top.html">
    <frameset cols="50%,50%">
        <frame src="left.html">
        <frame src="right.html">
    </frameset>
</frameset>

6. No Frames Fallback

For browsers that do not support frames, you can provide alternative content using the <noframes> tag.

<noframes>
    <p>Your browser does not support frames. Please update your browser.</p>
</noframes>

7. Example: Simple Website with Frames

Here’s a basic example of a simple website layout using frames:

<frameset rows="20%,*">
    <frame src="header.html" name="header">
    <frameset cols="20%,*">
        <frame src="navigation.html" name="nav">
        <frame src="content.html" name="content">
    </frameset>
</frameset>

8. Why Frames Are Obsolete

Frames have largely fallen out of favor due to several issues:

  • Accessibility problems for users with disabilities.
  • Complexity in bookmarking and navigating specific pages.
  • Increased difficulty in search engine indexing.
  • Modern alternatives like CSS layouts (Flexbox, Grid) provide better control and flexibility.

1. `<iframe>` Tag

The `<iframe>` (Inline Frame) tag is used to embed another HTML document within the current document. This allows you to display content from another source or webpage inside a defined area on your page.

Basic Syntax:

<iframe src="https://www.example.com" width="600" height="400" frameborder="0"></iframe>

Common Attributes:

  • src: The URL of the page to embed.
  • width and height: Specify the dimensions of the iframe.
  • frameborder: Specifies whether or not to display a border around the iframe (`0` for no border, `1` for border).

2. `<div>` Tag

The `<div>` tag is a block-level container used to group HTML elements and apply CSS styles or scripts to them. It’s often used to create layout sections in a webpage.

Basic Syntax:

<div class="content">
    <h2>Title</h2>
    <p>This is a paragraph inside a div block.</p>
</div>

Common Use:

Grouping content, creating layouts (e.g., headers, footers, sidebars), and applying styles with CSS.

3. `<font>` Tag

The `<font>` tag was historically used to define the font face, size, and color of text. However, it is deprecated in HTML5 in favor of CSS for styling text.

Basic Syntax:

<font size="3" color="blue" face="Arial">This is a sample text.</font>

Attributes:

  • size: Specifies the size of the font. Values range from `1` (smallest) to `7` (largest), or you can use relative values like `+1` or `-1` to increase or decrease size.
  • color: Sets the color of the font.
  • face: Specifies the font family (e.g., Arial, Times New Roman).
